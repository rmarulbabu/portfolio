# Arulbabu — Portfolio (React + Vite)

A premium, dark-futuristic single-page portfolio for a Python Full Stack Developer.
Built with React 18 + Vite, Tailwind CSS, Framer Motion, react-type-animation, react-parallax-tilt and react-icons.

Sections: Hero · About · Skills · Projects · Services · Testimonials · Contact · Footer.

## Quick start
\`\`\`bash
npm install
npm run dev
\`\`\`
Open http://localhost:3000.

## Build
\`\`\`bash
npm run build
npm run preview
\`\`\`

## Personalize
Edit src/data/portfolio.js to update profile, projects, services, testimonials.
Drop your photo into public/ and update PROFILE.photo.
Replace public/resume.pdf with your real resume.

## Optional backend
Copy .env.example to .env and set VITE_BACKEND_URL to enable real contact form submissions.
